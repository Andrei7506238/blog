# Andrei Notes

Hi, I’m a software developer with a passion for exploring technology, solving problems, and sharing insights. This blog is my space to post ideas I think are worth sharing—mostly tech-related, sometimes a bit beyond the code.

Think of this as a mix of practical knowledge, curiosity-driven experiments, and reflections on the collective intelligence of the developer community. Thanks for stopping by, and I hope you find something interesting here!

This is my personal blog powered by Jekyll and the Minimal Mistakes theme.

## Running Locally

If you'd like to run this environment locally to adapt it for your own use, follow these steps: 

0. **Prerequisites:**
   - Please check the [Jekyll installation guide](https://jekyllrb.com/docs/installation/) to ensure you have Ruby and Bundler set up.

1.  **Install Dependencies:**
    ```bash
    bundle install
    ```

2.  **Launch Server:**
    ```bash
    bundle exec jekyll serve
    ```
